Definisi

React.js atau React secara umum dapat dianggap sebagai pustaka JavaScript open-source yang digunakan untuk membuat user interface (UI) untuk aplikasi web. React JS hanya mengurusi semua hal yang berkaitan dengan tampilan dan logika di sekitarnya . Kerangka kerja yang mendasarinya kurang lebih sama dengan React tetapi alih-alih menggunakan komponen web, ia menggunakan komponen seluler asli. React dapat memiliki cara yang baik untuk menampilkan data yang terus berubah dalam sebuah aplikasi

Kegunaan

React JS adalah sebuah pustaka/library javascript yang bersifat open source untuk membangun User Interface yang dibuat oleh Facebook. React JS berfungsi untuk mendesain tampilan sederhana untuk setiap level dalam aplikasi, sehingga dapat digunakan untuk membuat dan mengembangkan pembuatan aplikasi berbasis web. React JS menjadi pilihan oleh developer-developer untuk mengembangkan aplikasi berbasis web dikarenakan ada banyak template-template yang bisa langsung dipakai secara langsung. Ada banyak keunggulan React JS yang bisa menjadi pillihan para developer, yaitu :

  • 1. React JS mendukung reusability komponen.
  • Mudah dalam menulis komponen.
  • Kinerja yang lebih baik karena virtual DOM, dll.


    • Performansi

    React JS memang sangat diminati developer website namun ada beberapa hal yang membuat react js lemot. sehingga diperlukan pengoptimalisasi react js. Beberapa cara mengoptimalkan react js, yaitu:

    • Pure Componen.
      Dengan melakukan pure componen setiap proses updating akan otomatis melakukan perbandingan state dan props saat ini dengan state dan props sebelumnya.
    • Memoization
      Memoization adalah proses yang berfungsi untuk menyimpan (cache) nilai dari fungsi yang melakukan komputasi yang berat sehingga ketika fungsi tersebut dipanggil kembali dengan agrumen yang sama maka akan mengembalikan nilai yang disimpan dengan tidak mengeksekusi ulang fungsi tersebut

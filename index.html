Tentang Defenisi

React.js atau React secara umum dapat dianggap sebagai pustaka JavaScript open-source yang digunakan untuk membuat user interface (UI) untuk aplikasi web. React JS hanya mengurusi semua hal yang berkaitan dengan tampilan dan logika di sekitarnya . Kerangka kerja yang mendasarinya kurang lebih sama dengan React tetapi alih-alih menggunakan komponen web, ia menggunakan komponen seluler asli. React dapat memiliki cara yang baik untuk menampilkan data yang terus berubah dalam sebuah aplikasi
